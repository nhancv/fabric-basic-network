# Usage

```bash
./scripts/bootstrap.sh [version] [ca version] [thirdparty_version]

Ex: 
./scripts/bootstrap.sh 1.2.1 1.2.1 1.2.1

# Start network
./startFabcar.sh

# Stop network
./stopFabcar.sh
```
